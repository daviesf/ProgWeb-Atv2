Visão Geral da Web

Colocar em perspectiva a programação para Web; compreender o funcionamento de aplicações baseadas na arquitetura cliente-servidor.

Marcação e Estilo

Capacitar o aluno a compreender as principais linguagens de marcação (HTML) e estilo (CSS); criar páginas web estáticas e bem estruturadas.

Scripts Client-Side

Capacitar o aluno a compreender e desenvolver scripts para execução em navegadores (front-end); utilizar JavaScript para criar páginas dinâmicas.

Scripts Server-Side

Capacitar o aluno a compreender e desenvolver scripts para execução em servidores web (back-end); permitir a criação de aplicações web completas.
